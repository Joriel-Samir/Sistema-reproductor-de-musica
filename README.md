# Sistema-reproductor-de-musica

Reproductor de música web estilo Spotify desarrollado con Flask y JavaScript vanilla, que implementa listas enlazadas circulares para la gestión de playlists y navegación automática de canciones.

Tecnologías
Backend

Python 
Flask - Framework web
Werkzeug - Manejo seguro de archivos

Frontend

JavaScript (Vanilla)
HTML5 
CSS3

Estructuras de Datos

Lista Enlazada Circular - Gestión de playlists y navegación
Algoritmo de Ordenamiento por Inserción - Ordenar canciones

Instalación
Requisitos previos

Python 3.7 o superior
pip (gestor de paquetes de Python)

Pasos de instalación

Clonar el repositorio

bashgit clone https://github.com/Joriel-Samir/Sistema-reproductor-de-musica.git
cd Sistema-reproductor-de-musica

Crear entorno virtual (recomendado)

bashpython -m venv venv

Activar entorno virtual

Windows:
bashvenv\Scripts\activate
macOS/Linux:
bashsource venv/bin/activate

Instalar dependencias

bashpip install flask

Ejecutar la aplicación

bashpython musicplayer/Musicplayerr/reproductor/server.py

Abrir en el navegador

http://127.0.0.1:5000
